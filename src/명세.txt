-"java 뱀사다리" 형태로 수행
-"시작"을 입력하면 2~6명 사이의 플레이어 수를 입력받아 숫자에 해당하는 사용자 명을 입력받아,
 첫 번째 플레이어부터 플레이 시작
-"roll"을 입력하면 해당 플레이어의 입력이 맞는지 확인 -> x번 플레이어님 돌리시겠습니까?
    -"Y"를 입력하면 해당 플레이어에 해당하는 주사위수를 생성하고 그 만큼 이동
    -"N"를 입력하면 다시 해당 플레이어의 "roll" 입력을 기다림
-"현재위치"를 입력하면 플레이어의 현재 칸 위치를 표시
-"조사 플레이어명" 형태로 입력하면 입력된 플레이어가 이동 가능한 앞 6칸의 정보 표시(사다리, 뱀여부 표시)
-모든 사용자가 도착하면 게임 종료
-"종료"를 입력하면 프로그램 종료
-서버/클라이언트 형태로 구성하여 사용자가 각자 컴퓨터에서 수행할 수 있도록 구현하면 가산점 부여

명령어
roll: 주사위를 던지다
checkInfo player_name: player_name 의 이동가능한 앞 6칸의 정보 표시
pos: 위치 출력
exit:
